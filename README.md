# Orbit - Simple note taking app

Working on progress project...

## Usage

1. Clone project

```
git clone https://github.com/nghiaxh/orbit.git
```

2. Install dependencies

```
cd backend
npm install     # or yarn install

cd frontend
npm install     # or yarn install
```

3. Config environment variable

- Create `.env` at `ebookshelf/backend/`

- Add `MONGODB_URI`

```env
MONGODB_URI = "mongodb://localhost:27017/orbit"
```

4. Run both `backend` and `frontend`folder

```
cd backend
npm run start   # or yarn start

cd frontend
npm run dev     # or yarn dev
```

5. Access application at `http://localhost:3001`
